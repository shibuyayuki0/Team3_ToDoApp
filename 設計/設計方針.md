## 画面・C#クラス・インターフェース定義、アプリケーション設定
- Program.cs
  - DI配線などを行う

- _Layout.cshtml
  - アプリの外見を決める
  
- Priorities.cs
  - 優先度設定を表すレコード
  - prioritiesテーブルに対応
  - int Id　・・・　優先度ID
  - int Name　・・・　優先度ラベル
    
- Tasks.cs
  - タスク1件を表すレコード
  - tasksテーブル中の1件レコード（1行）に対応
  - int TaskId　・・・　タスクのID
  - string TaskName　・・・　タスク名
  - string ContentText　・・・　タスクの詳細内容
  - DateTime DeadlineAt　・・・　締切日
  - int PriorityId　・・・　優先度ID
  - DateTime CompletedAt　・・・　完了日
 
- Index.cshtml / Index.cshtml.cs
  - アプリのホーム画面、タスク一覧の表示、フィルタリング、詳細表示を行う
  - タスクの編集・新規作成画面を呼び出す
  - string FilterName　・・・　フィルター状態をリダイレクト時に引き継ぐプロパティ
  - int SelectedId　・・・　現在選択中のタスクId
  - List<Tasks> TaskList　・・・　タスクのプレビュー一覧を保持するプロパティ
  - List<Priorities> PriorityList　・・・　優先度の選択肢を表すプロパティ
  - void OnGet()　・・・　ページロード時の処理、TaskListの初期化を行う
  - void OnGetFiltering()　・・・　フィルターを切り替えて再度リストを更新する
  - IActionResult OnPostDelete() ・・・　指定したタスクのDeletedAtを更新して、Indexにリダイレクトする
  - IActionResult OnPostComplete()　・・・　タスクの完了ボタンに対応、指定したタスクのCompletedAtを更新して、Indexにリダイレクトする
  
- ToDoEditor.cshtml / ToDoEditor.cshtml.cs
  - タスクの編集、新規作成を行う
  - int? TargetTaskId　・・・　編集対象のタスクIdを表すプロパティ、nullなら新規作成
  - Tasks?　TargetTask　・・・　編集対象のタスク本体を表すプロパティ、SupportGet = trueを付けることでGETで受け取れる
  - string TaskName　・・・　タスク名を表すプロパティ
  - List<Priorities> PriorityList　・・・　優先度の選択肢を表すプロパティ
  - int SelectedPriorityId　・・・　選択された優先度のIdを表すプロパティ
  - DateTime DeadlineAt　・・・　締切日を表すプロパティ
  - string ContentText　・・・　タスクの詳細内容を表すプロパティ
  - void OnGet()　・・・　TargetTaskIdがnullでないときは、TargetTaskを取得し、中身を各プロパティの初期値として表示する
                          新規作成のときは、DeadlineAtに今日の日付を初期値として入れる
  - IActionResult OnPost()　・・・　指定したIDまたは新規に基づくタスクをDBに登録し、Indexにリダイレクトする

- ListFilterMode.cs
  - フィルターの設定を表すenum
  - Continue = 0, Complete, All

- ITaskDataEditor.cs
  - タスクデータを編集・管理するインターフェース
  - void Add(Tasks task)　・・・　taskを受け取って、DBに追加する
  - List<Tasks> GetTaskList(ListFilterMode mode)　・・・　modeの条件に従って、tasksテーブルを取得する
  - List<Priorities> GetPriorityList()　・・・　prioritiesテーブルを取得する
  - Tasks GetTask(int id)　・・・　tasksテーブルから、指定IDのレコードを取得する
  - void Update(Tasks task)　・・・　taskを受け取って、DBを更新する
  - void Delete(int id)　・・・　tasksテーブルから、指定したidのレコードを削除する

- TaskDataEditor.cs
  - ITaskDataEditorの実装


## Razor Pages制御文
- Index.cshtml
  - @foreach(var task in TaskList)　・・・　TaskListを1行づつに分解し、Task1件の定義を行う
  - @switch(task.PriorityId)　・・・　タスクの優先度に応じて、タスク1件ごとの色を変える
- ToDoEditor.cshtml
  - @foreach(var priority in PriorityList)　・・・　PriorityListを1行づつに分解し、優先度のドロップダウンリストの定義を行う

## JavaScript機能・定義
### 主な機能
- Index.cshtml（メイン側）
  - @Json.Sirializer(Model.TaskList)　・・・　TaskListをPageModelからJson形式で受け取る、プロパティはcamelCaseになる
  - @Json.Sirializer(Model.Priorities)　・・・　PrioritiesをPageModelからJson形式で受け取る
  - 詳細メニュー表示　・・・　タスク1件を押すと.activeで開閉、タスク1件ごとのdata-id="@task.taskId"を振り、これを詳細画面に渡す
  - 完了したタスク　・・・　task.completedAtで完了時の見た目に変える
  - トップに戻るボタン　・・・　スクロール量で.active切替で表示、#に飛ぶ
  - 以下はチャレンジ課題の領域なので、追加実装で想定しておく
    - 遅れマーク表示　・・・　タスク1件ごとにtask.deadlineAtと今日の日付を比較して判定、.activeの有無で表示
    - タスクの残り時間/遅れ時間表示　・・・　比較した差分を表示、0以上なら”残り”、0未満なら"遅れ"に変える、色変更も行う
- Index.cshtml（詳細メニュー側）
  - 詳細メニュー表示中は、メニュー外暗転＆暗転部クリックメニュー閉じる
  - ×ボタン　・・・　自身の.activeを切り替える
  - 優先度　・・・　task.priorityIdに応じて、”優先度：＊”の表記と書式を切り替える
  - priority　・・・　優先度表示する文字列
  - taskName　・・・　タスク名を表す文字列、task.taskNameを使う
  - deadlineAt　・・・　締切日を表す文字列、task.deadlineAtを使う
  - contentText　・・・　詳細内容を表す文字列、task.contentTextを使う
  - 編集ボタン　・・・　詳細メニュー表示時にhrefを /ToDoEditor?id=**　のように書き換える
  - 以下はJSと直接の関係は薄いが表記
    - 完了ボタン　・・・　SelectedIDを完了状態にする（Post発行）
    - 削除ボタン　・・・　SelectedIDを削除状態にする（Post発行）

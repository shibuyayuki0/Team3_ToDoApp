# 設計方針

## テーブルイメージ
| タスクID | タスク名 | 詳細内容 | 期限 | 優先度 | 完了日 | 作成日 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| id | task_name | content | deadline_at | priority | completed_at | created_at |

## 画面イメージ
### Index
- 一覧方式で、上から下に流れるように配置
  - 色で重要度判別
  - 完了ボタン、タスク名、締切日（あと何日）で表示
  - 〆切オーバーは△！マークで表現
- タスク1行を押すと右メニュー展開し詳細を表示
  - 右メニュー内に、重要度、タスク名、締切日を表示
  - 編集ボタン（青系）、削除ボタン（赤・白抜き）も配置
- タスク行の下部に作成ボタンをフロート表示
  - 作成

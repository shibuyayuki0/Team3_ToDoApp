# 設計方針
## 役割分担
- タスクの一覧部分 詳細メニュー　・・・　渋谷
- 更新　新規作成　・・・　福山
- 画面全体（シェル）　DBサポート　・・・　越智

## テーブルイメージ
### prioritiesテーブル
| 名称 | カラム名 | データ型 | 幅 | PK | UNIQURE | NOT NULL | FK |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| ID | priority_id | int | 4バイト | TRUE |←|←| - |
| 優先度 | priority_name | nvarchar | 1文字 | - | - | TRUE | - |

### tasksテーブル
| 名称 | カラム名 | データ型 | 幅 | PK | UNIQURE | NOT NULL | FK |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|タスクID|id|int|4バイト|TRUE|←|←|-|
|タスク名|task_name|nvarchar|30文字|-|-|TRUE|-|
|詳細内容|content|nvarchar|255文字|-|-|TRUE|-|
|期限|deadline_at|datetime|日時|-|-|TRUE|-|
|優先度|priority_id|int|4バイト|-|-|TRUE|priority|
|完了日|completed_at|datetime|日時|-|-|-|-|
|作成日|created_at|datetime|日時|-|-|TRUE|-|
|削除日|deleted_at|datetime|日時|-|-|-|-|


## 画面・C#クラス・インターフェース定義、アプリケーション設定
- Program.cs
  - DI配線などを行う

- _Layout.cshtml
  - アプリの外見を決める
  
- Priorities.cs
  - 優先度設定を表すレコード
  - prioritiesテーブルに対応
    
- Tasks.cs
  - タスク1件を表すレコード
  - tasksテーブル中の1件レコード（1行）に対応
 
- Index.cshtml / Index.cshtml.cs
  - アプリのホーム画面、タスク一覧の表示、フィルタリング、詳細表示を行う
  - タスクの編集・新規作成画面を呼び出す
  - [field]ITaskDataEditor _taskDataEditor　・・・　
  - string StartFilter　・・・　サーバー側でフィルターの初期値を記憶するプロパティ
  - List<Tasks> TaskList　・・・　タスクのプレビュー一覧を保持するプロパティ
  - int SelectedId　・・・　現在選択中のタスクId
  - void OnGet()　・・・　ページロード時の処理、TaskListの初期化を行う
  - IActionResult OnPostDelete() ・・・　指定したタスクのDeletedAtを更新して、Indexにリダイレクトする
  - IActionResult OnPostComplite()　・・・　タスクの完了ボタンに対応、指定したタスクのComplitedAtを更新して、Indexにリダイレクトする
  
- ToDoEditor.cshtml / ToDoEditor.cshtml.cs
  - タスクの編集、新規作成を行う
  - int? TargetTaskId　・・・　編集対象のタスクIdを表すプロパティ、nullなら新規作成
  - Tasks?　TargetTask　・・・　編集対象のタスク本体を表すプロパティ
  - string TaskName　・・・　タスク名を表すプロパティ
  - List<Priorities> PriorityList　・・・　優先度の選択肢を表すプロパティ
  - int SelectedPriorityId　・・・　選択された優先度のIdを表すプロパティ
  - DateTime DeadlineAt　・・・　締切日を表すプロパティ
  - string ContentText　・・・　タスクの詳細内容を表すプロパティ
  - void OnGet()　・・・　TargetTaskIdがnullでないときは、TargetTaskを取得し、中身を各プロパティの初期値として表示する
                    新規作成のときは、DeadlineAtに今日の日付を初期値として入れる
  - IActionResult OnPost()　・・・　指定したIDまたは新規に基づくタスクをDBに登録し、Indexにリダイレクトする

- ITaskDataEditor.cs
  - タスクデータを編集・管理するインターフェース
  - void Add(Tasks task)　・・・　taskを受け取って、DBに追加する
  - List<Tasks> GetTaskList()　・・・　tasksテーブルを取得する
  - List<Priorities> GetPriorityList()　・・・　prioritiesテーブルを取得する
  - Tasks GetTask(int id)　・・・　tasksテーブルから、指定IDのレコードを取得する
  - void Update(Tasks task)　・・・　taskを受け取って、DBを更新する
  - void Delete(int id)　・・・　tasksテーブルから、指定したidのレコードを削除する

- TaskDataEditor.cs
  - ITaskDataEditorの実装

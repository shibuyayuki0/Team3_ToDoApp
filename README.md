# チーム３　チームワーク課題：簡易ToDoアプリ
本システムは、チーム進捗などのタスクを名称・優先度・期日で管理するためのWEBアプリケーションです。<br/>
タスクを一覧で表示し、タスクの状態を未完了・完了別に管理することができます。<br/>

# 機能

## 基本機能
- タスクを一覧で表示する
- タスクの状態（「すべて」・「未完了」・「完了」）でフィルター表示
- 1件のタスクに対し、タスク名・期限・詳細内容・優先度を登録
- 既存のタスクを編集・更新・削除が可能
- 一覧画面上でワンクリックでタスクの完了⇔未完了の切り替えが可能

## 追加の機能
- タスクの優先度別に色分けをして表示
- タスクの期日を昇順で表示する（期日が近いものが一番上に表示）
- タスクにディレイがあると、期日の横にアラート表示
